## Mind Meetme 

### Source Frontend (User)

### How to run this project?

#### 1. Clone project

```bash
$ git clone https://github.com/levanthuan2001qttl/Mind_fronend
```

#### 2. Change directory to folder frontend

```bash
$ cd Mind_fronend
```

#### 3. Install dependencies

```bash
$ npm install
```

#### 4. Run in developement mode

```bash
$ npm start
```
